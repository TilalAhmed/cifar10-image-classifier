# CIFAR-10 Image Classifier 🖼️

A deep learning project that classifies images into 10 categories using a Convolutional Neural Network (CNN).

---

## Overview

Started with an ANN baseline model achieving 49% accuracy, then built a CNN architecture 
using TensorFlow/Keras that improved accuracy to 68.9% on the CIFAR-10 dataset.

---

## Categories

Airplane · Automobile · Bird · Cat · Deer · Dog · Frog · Horse · Ship · Truck

---

## Results

| Model | Accuracy |
|-------|----------|
| ANN (baseline) | 49% |
| CNN | 68.9% |

---

## Tech Stack

- **Language:** Python
- **Libraries:** TensorFlow · Keras · NumPy · Matplotlib
- **Dataset:** CIFAR-10 (built into TensorFlow)

---

## Dataset

CIFAR-10 is built into TensorFlow and loads automatically when you run the notebook.
No download needed.
